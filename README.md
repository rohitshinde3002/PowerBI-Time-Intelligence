# 📊 Power BI Time Intelligence Dashboard

## 📌 Project Overview

This project is based on **Power BI Time Intelligence** concepts.
In this dashboard, I have created interactive data visualizations to analyze sales performance over time.

The goal of this project is to understand:

* Year-wise Sales Analysis
* Month-wise Trends
* Growth Comparison (YoY – Year over Year)
* KPI Indicators
* Performance Tracking

---

## 🛠 Tools Used

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Time Intelligence Functions

---

## 📈 Key Features

✅ KPI Cards (Total Sales, Total Profit, Growth %)
✅ Monthly & Yearly Sales Trend Charts
✅ Year-over-Year Comparison
✅ Interactive Filters (Slicers)
✅ Clean and Professional Dashboard Layout

---

## 📚 Concepts Applied

* CALCULATE()
* SAMEPERIODLASTYEAR()
* DATEADD()
* TOTALYTD()
* Time-based Filtering
* Measures & Calculated Columns

---

## 🎯 Project Objective

The main objective of this project is to practice and demonstrate:

* Data Visualization Skills
* Business Intelligence Concepts
* Time-based Data Analysis
* Dashboard Design
